Welcome to JOE

Welcome On JOE System. Java Online Examination System project could be a web portal which is developed or implemented in java domain or platform. This project is helpful for students to practice different java mock examinations from this site. In current generation lots of the examinations are conducted through online system. This project will help students to get practiced to online examination method by taking mock tests from this web portal. Java Online Examination System portal is implemented in 2 modules::
1) Student (User) examination module, and
2) Admin module.
Admin module will add multiple Java questions according to different level, so students can easily take test step-by-step. In Student examination module, students as a User are required to register themselves, which is mandatory, after which they are able to participate in the online test. GOOD LUCK
